## 2. Рейтинг - загрузка XLSX ✅

**Как работает:**
1. Открываешь `/rating`
2. Нажимаешь **"Показать таблицу"**
3. XLSX загружается
4. Отображается таблица со студентами

**Если ошибка:**
- Показывается кнопка **"Открыть файл напрямую"**
- Файл откроется в новой вкладке

## 3. Все ссылки отображаются ✅

**Было:** Видна только 1 ссылка  
**Стало:** Все ссылки из описания показываются списком!

**Пример:**
```
ДЗ
├─ Упражнения 1-5
├─ [Скачать файл 1] ← ссылка 1
├─ [Скачать файл 2] ← ссылка 2
└─ [Скачать файл 3] ← ссылка 3
```

Функция `extractAllLinks()` находит все `<a href>` в HTML и показывает их все!

## 4. Полноценный редактор ДЗ ✅

**Создание записи:**
1. Нажимаешь **"Добавить ДЗ"**
2. Вводишь дату
3. Выбираешь группу (B1, B2, МБС-07, МПБ-06)
4. Создаёшь

**Добавление задания:**
1. Выбираешь запись слева
2. Нажимаешь **"Добавить задание"**
3. Заполняешь:
   - Тип (На паре/ДЗ/Долг/Рейтинг)
   - Название
   - Описание (текст)
   - Ссылка на файл
4. Сохраняешь

**Редактирование:**
- Нажимаешь ✏️ на записи → меняешь дату/группу
- Нажимаешь ✏️ на задании → меняешь текст/ссылку
