# 🔧 Исправления

## ✅ Что исправлено

### 1. Падение домашнего задания в админке
**Ошибка:** `can't access property "length", entry.tasks is undefined`

**Исправление:**
```tsx
// Было:
{entry.tasks.length}

// Стало:
{(entry.tasks || []).length}
```

Добавлена проверка на `undefined` для всех обращений к `entry.tasks`.

---

### 2. Полукруги в дэшборде ИСПРАВЛЕНЫ НАКОНЕЦ-ТО

**Было:** Кривые круги через CSS  
**Стало:** Аккуратный quarter-circle через SVG

```tsx
<svg className="absolute top-0 right-0 w-32 h-32" viewBox="0 0 128 128">
  <path d="M128 0C128 70.689 70.689 128 0 128V0H128Z" opacity="0.3" />
</svg>
```

Теперь это ровный закруглённый угол в правом верхнем углу каждой карточки!
