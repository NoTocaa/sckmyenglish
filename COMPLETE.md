## 2. Рейтинг - визуальное отображение XLSX ✅

**Как работает:**
1. Открываешь `/rating`
2. Видишь список файлов рейтинга
3. Нажимаешь **"Показать таблицу"**
4. XLSX файл загружается и отображается
5. Видишь таблицу со студентами и баллами

**Таблица показывает:**
- Позицию в рейтинге
- Имя студента
- Баллы за ДЗ
- Баллы за работу на паре
- Общий балл
- Статус (OK/Долг)

**Пример:**
```
┌────┬───────────────┬─────┬────────┬───────┬────────┐
│ #  │ Студент       │ ДЗ  │ На паре│ Всего │ Статус │
├────┼───────────────┼─────┼────────┼───────┼────────┤
│ 1  │ Иванов Иван   │ 45  │ 40     │ 85    │ OK     │
│ 2  │ Петров Петр   │ 40  │ 38     │ 78    │ OK     │
│ 3  │ Сидоров Сидор │ 30  │ 25     │ 55    │ Долг   │
└────┴───────────────┴─────┴────────┴───────┴────────┘
```

## 3. Полукруги в админке ИСПРАВЛЕНЫ ✅

**Было:** SVG с разрезанными кругами  
**Стало:** Чистый градиент без лишних элементов

```tsx
<div className="absolute inset-0 bg-gradient-to-br from-white/5 to-transparent" />
```

Теперь карточки статистики выглядят чисто и аккуратно!
