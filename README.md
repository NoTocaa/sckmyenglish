# SckMyEnglish

Современная платформа для изучения английского языка в СГУ. Сайт создан как альтернатива nsbonline.ru с улучшенным дизайном, удобным интерфейсом и автоматическим парсингом домашних заданий.

## 🚀 Особенности

- **Современный дизайн** — тёмная тема, градиенты, адаптивный интерфейс
- **Админ-панель** — удобное управление домашними заданиями и материалами
- **Авто-парсинг** — возможность автоматического обновления ДЗ с nsbonline.ru
- **Типизация** — полный TypeScript с правильными типами
- **Модульная архитектура** — чистая структура проекта

## 🛠 Установка и запуск

### Требования

- Node.js 18+ 
- npm/pnpm

### Установка зависимостей

```bash
npm install
```

### Запуск разработки

```bash
npm run dev
```

### Сборка продакшена

```bash
npm run build
```

## 📄 Страницы

### Публичные

| Страница | Маршрут | Описание |
|----------|---------|----------|
| Главная | `/` | Hero секция, карточки функций |
| Ответы | `/answers` | Список домашних заданий по датам |
| Полезное | `/useful` | Учебные материалы и файлы |

### Админ-панель

| Страница | Маршрут | Описание |
|----------|---------|----------|
| Дашборд | `/admin` | Общая статистика, быстрые действия |
| ДЗ | `/admin/homework` | Управление домашними заданиями |
| Материалы | `/admin/useful` | Управление файлами |
| Настройки | `/admin/settings` | Конфигурация сайта, парсер |

## 🔧 Парсинг с nsbonline.ru

### Одноразовый парсинг

```bash
npm run parse
```

### Фоновый режим (каждые 6 часов)

```bash
npm run parse:watch
```

### Настройка парсера

1. Откройте `scripts/parse-homework.ts`
2. Обновите CSS селекторы под актуальную структуру nsbonline.ru
3. Запустите парсер

> **Примечание:** Так как nsbonline.ru не имеет защиты от роботов и SSL, парсинг работает напрямую.

## 📝 Добавление контента

### Через админ-панель (рекомендуется)

1. Перейдите в `/admin`
2. Выберите нужный раздел
3. Добавьте/отредактируйте контент

### Через код

Отредактируйте файлы в `src/app/data/`:
- `homeworkData.ts` — домашние задания
- `usefulData.ts` — полезные файлы
