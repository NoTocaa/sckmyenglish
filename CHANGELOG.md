# Changelog

Все изменения в проекте sckmyenglish.

## [1.0.0] - 2026-03-19

### ✨ Новое

#### Рефакторинг
- Вынесены все данные в отдельные файлы (`homeworkData.ts`, `usefulData.ts`)
- Созданы TypeScript типы для всех сущностей
- Добавлен кастомный хук `useHomeworkData` для управления состоянием
- Создан переиспользуемый компонент `HomeworkList`

#### Главная страница
- Полностью переработанный дизайн с hero-секцией
- Градиентные фоны и современные карточки
- Призывы к действию (CTA кнопки)
- Секция с преимуществами
- Информационный блок

#### Админ-панель
- **Дашборд** — статистика, быстрые действия, последние ДЗ
- **Управление ДЗ** — добавление, редактирование, удаление заданий
- **Управление материалами** — загрузка полезных файлов
- **Настройки** — конфигурация сайта и парсера
- Адаптивный дизайн с боковой панелью
- Мобильное меню

#### Парсер
- Скрипт для парсинга nsbonline.ru
- Планировщик для автоматического обновления (каждые 6 часов)
- Создание бэкапов перед обновлением
- Конфигурируемые интервалы

#### Дополнительные компоненты
- `Footer` — подвал сайта с контактами
- `SearchBar` — компонент поиска
- `LoadingSpinner` — индикатор загрузки

#### Документация
- Полный README с описанием всех страниц
- Инструкция по установке и развёртыванию
- Документация по парсеру
- Changelog

### 🔧 Технические изменения

#### Зависимости
- Добавлен `tsx` для выполнения TypeScript скриптов
- Добавлен `@types/node` для типизации Node.js API

#### Скрипты
- `npm run parse` — запуск парсера
- `npm run parse:watch` — фоновый режим парсера

#### Структура
```
src/app/
├── components/
│   ├── Footer.tsx
│   ├── SearchBar.tsx
│   ├── LoadingSpinner.tsx
│   └── HomeworkList.tsx
├── data/
│   ├── homeworkData.ts
│   └── usefulData.ts
├── hooks/
│   └── useHomeworkData.ts
├── pages/
│   └── admin/
│       ├── AdminLayout.tsx
│       ├── AdminDashboard.tsx
│       ├── AdminHomework.tsx
│       ├── AdminUseful.tsx
│       └── AdminSettings.tsx
└── types/
    └── homework.ts

scripts/
├── parse-homework.ts
└── scheduler.ts
```

### 🎨 Улучшения дизайна

- Обновлены страницы Answers и Useful
- Добавлены иконки и hover-эффекты
- Улучшена типографика
- Консистентная цветовая схема

### 📱 Адаптивность
