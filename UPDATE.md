## 🚀 Как использовать

### Запуск сайта
```bash
npm run dev
```

### Парсинг ДЗ
```bash
# 1. Открой scripts/parse-homework.ts
# 2. Измени GROUP_PAGE на свою группу
# 3. Запусти
npm run parse
```

### Доступные группы
- `group_b1_fkn1_02_2026.html` - B1
- `group_b2_fkn1_02_2026.html` - B2
- `group_mbs07_fkn3_02_2026.htm` - МБС07
- `group_mpb06_fkn3_02_2026.html` - МПБ06
