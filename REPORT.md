## 🚀 Как использовать

### Запуск проекта

```bash
npm install
npm run dev
```

### Доступ к страницам

- Главная: http://localhost:5173/
- Ответы: http://localhost:5173/answers
- Полезное: http://localhost:5173/useful
- Админка: http://localhost:5173/admin

### Редактирование контента

1. Откройте `/admin`
2. Выберите нужный раздел
3. Добавьте/отредактируйте контент

Или редактируйте файлы в `src/app/data/`
