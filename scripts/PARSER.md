# Настройка парсера nsbonline.ru

## ✅ Парсер работает!

Парсер успешно подключается к nsbonline.ru и забирает данные.

## 📁 Как использовать

### 1. Выберите вашу группу

Откройте `scripts/parse-homework.ts` и найдите строку:

```typescript
const GROUP_PAGE = 'group_b1_fkn1_02_2026.html'; // Change to your group
```

Замените на вашу группу. Доступные группы (смотрите на nsbonline.ru):

**2 семестр 2026:**
- `group_b1_fkn1_02_2026.html` - Группа B1
- `group_b2_fkn1_02_2026.html` - Группа B2
- `group_mbs07_fkn3_02_2026.htm` - Группа МБС07
- `group_mpb06_fkn3_02_2026.html` - Группа МПБ06

**1 семестр 2025:**
- `group_b1_fkn1_01_2025.html`
- `group_b2_fkn1_01_2025.html`
- `group_b_fkn2_2025.html`
- `group_mbs07_fkn3_01_2025.htm`
- `group_mpb06_fkn3_01_2025.htm`

### 2. Запустите парсер

```bash
npm run parse
```

### 3. Проверьте результат

Откройте `src/app/data/homeworkData.ts` - там будут новые данные.

### 4. Запустите сайт

```bash
npm run dev
```

Откройте http://localhost:5173/answers и проверьте ДЗ.

## 🔧 Решение проблем

### Парсер не находит страницу

1. Проверьте, что группа существует на nsbonline.ru
2. Откройте главную страницу http://nsbonline.ru
3. Найдите вашу группу в списке
4. Скопируйте точное название файла

### Неправильная кодировка

Парсер автоматически декодирует windows-1251 → UTF-8. Если видите кракозябры:
- Проверьте `debug-group.html` в папке `scripts/`
- Отправьте файл разработчику

### Ссылки не работают

Парсер автоматически добавляет `http://nsbonline.ru/` к относительным ссылкам.
Если файлы не открываются, проверьте, что они существуют на сайте.

Found 50 potential homework links
