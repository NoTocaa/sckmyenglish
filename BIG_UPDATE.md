### 4. Динамический рейтинг 🏆

**Новая страница:** `/rating`

**Возможности:**
- ✅ **Авто-обновление каждые 30 секунд** - не нужно перезагружать
- ✅ **Таблица со всеми студентами** - отсортирована по баллам
- ✅ **Статусы:** 
  - 🟢 OK - без долгов
  - 🔴 Долг - есть задолженность
- ✅ **Разделение баллов:**
  - Домашние задания
  - Работа на паре
  - Общий балл
- ✅ **Позиции в рейтинге** - кто на каком месте
- ✅ **Кнопка обновления** - вручную обновить данные
- ✅ **Скачивание оригинала** - ссылка на файл

**Как работает:**
1. Парсер находит файл с рейтингом (points_comp*.xlsx или similar)
2. Компонент RatingTable загружает его
3. Данные парсятся и отображаются в таблице
4. Каждые 30 секунд происходит авто-обновление

**Статистика на странице:**
- Количество студентов
- Средний балл
- Количество должников

## 📝 Как добавить рейтинг

1. **Парсер найдёт автоматически:**
   - Если на nsbonline.ru есть файл со словом "рейтинг", "points", "rating"
   - Он автоматически появится на странице /rating

2. **Вручную через админку:**
   - Открой /admin/homework
   - Добавь ДЗ
   - В вопросе укажи "Рейтинг"
   - В ответе ссылка на файл

## 💡 Советы

1. **Проверяй тип задания:**
   - "На паре" - делается на уроке

## 🔧 Технические детали

### Компоненты
- `RatingTable.tsx` - таблица рейтинга с авто-обновлением
- `Answers.tsx` - страница заданий с группировкой
- `Root.tsx` - навигация с выбором страниц

### Типы
```typescript
type TaskType = "in_class" | "homework" | "debt" | "rating";
```

### Парсинг типов
```typescript
- "рейтинг", "points" → rating
- "долг", "не был" → debt
- "на паре", "в классе" → in_class
- остальное → homework
```
