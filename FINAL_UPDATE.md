### 2. Полная переработка структуры данных ✅

**Было:**
```typescript
// Старая структура
{
  id: number;
  date: string;
  answers: Array<{ question, answer }>;
}
```

**Стало:**
```typescript
// Новая, правильная структура
{
  id: number;
  date: string;           // "19.03.2026"
  group: string;          // "mbs07", "mpb06" - для какой группы
  tasks: Task[];          // Массив заданий
  note?: string;          // Общая заметка на день
}

// Задание
interface Task {
  id: number;
  question: string;       // "Задание 1", "F1"
  description: string;    // Текстовое описание (понятно!)
  fileUrl?: string;       // Ссылка на файл
  fileLabel?: string;     // "Скачать TM tools"
  type: TaskType;         // "in_class" | "homework" | "debt" | "rating_file"
  answer: string;         // HTML для совместимости
}
```

**Преимущества:**
- ✅ Понятная структура
- ✅ Разделение текста и файлов
- ✅ Поддержка разных групп
- ✅ Типы заданий

### 6. Рейтинг - персональные баллы ✅

**Понял правильно:**
- Это НЕ соревнование между студентами
- Это персональные баллы каждого
- Нужен для отслеживания успеваемости
- Если баллов нормально → меньше проблем на сессии

**Файл рейтинга:**
- Лежит на nsbonline.ru/группа
- Называется типа `points_comp_b2_2term_26.xlsx`
- Ссылка в HTML: `<a href="files/points_comp_b2_2term_26.xlsx">рейтинг</a>`
- Может отсутствовать в некоторые дни (как 19.03)

**Как работает:**
1. Парсер находит файл со словом "рейтинг" или "points"
2. Помечает задание как тип `rating_file`
3. На странице заданий оно показывается с иконкой 📊
4. Можно скачать и посмотреть свои баллы

## 🔧 Технические детали

### Хранение данных

```typescript
// localStorage
selectedGroup: "mbs07"  // Выбор студента
```

### Типы заданий
```typescript
type TaskType = "in_class" | "homework" | "debt" | "rating_file";
```

### Автоматическое определение типа
```typescript
function determineTaskType(question: string, answer: string): TaskType {
  const text = `${question} ${answer}`.toLowerCase();
  
  if (text.includes('рейтинг') || text.includes('points')) 
    return "rating_file";
  if (text.includes('долг') || text.includes('не был')) 
    return "debt";
  if (text.includes('на паре') || text.includes('устно')) 
    return "in_class";
  return "homework";
}
```
